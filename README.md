# Каталог путешествий

React-приложение для каталогизации путешествий.

## Функциональность
- Просмотр карточек путешествий
- Фильтрация по странам
- Добавление новых путешествий
- Лайки для каждого путешествия

## Установка и запуск
1. Клонировать репозиторий
2. `npm install`
3. `npm run dev`

## Технологии
- React
- Vite
- CSS Modules
